# Manual de Usuario - SHOESKI  
*Calzado bien hecho*

## Introducción  

### Objetivo del manual  
Este manual proporciona una guía completa sobre el uso de todas las funcionalidades de nuestro sitio web. Su objetivo es explicar claramente el propósito de cada botón, pantalla y proceso, permitiendo a los usuarios resolver dudas de manera autónoma.  

### Descripción del sitio web  
**SHOESKI** es una tienda en línea especializada en calzado para toda la familia. Ofrecemos:  
- Amplia variedad de modelos (mujer, hombre)  
- Estilos casuales, deportivos, escolares y de vestir  
- Catálogo completo con filtros por genero 
- Envíos a nivel nacional  
- Métodos de pago seguros  
- Atención personalizada  

Nuestra misión es brindar comodidad, estilo y calidad mediante productos seleccionados y marcas reconocidas.

### Funciones administrativas  
Acceso exclusivo para personal autorizado:  
1. **Gestión de productos**:  
   - Agregar nuevos artículos  
   - Actualizar detalles existentes  
   - Eliminar productos discontinuados  
2. **Gestión de usuarios**:
   - Agregar nuevos usuarios
   - Editar o eliminar registros  
   - Mantener información actualizada
3. **Gestión de proveedores**:
   - Agregar nuevos proveedores
   - Editar o eliminar registros  
   - Mantener información actualizada  

**Objetivo**: Garantizar precisión en el catálogo y evitar confusiones.  


**Política de privacidad**:  
Toda la información se almacena con protocolos de seguridad avanzados.
